# MISSPELLER

Module that makes typos.

Works with Python 3.

## Installation
    pip install misspeller

## Usage
    >>> import misspeller

    >>> misspeller.change_text('Why so serious?')
    'Whg so srriojs?'

    >>> misspeller.change_word('hello')
    'yello'

    >>> misspeller.change_symbol('b')
    'n'
